void devio_init ( dev_io_ops_t *  io)

Initialise device IO manager state.

This function sets the function pointers for the device IO interfaces that any libplatsupport driver depends on. This means that this initialisation must be done before any libplatsupport drivers can be used.

Parameters
ioThe device state structure to initialise (No ownership transfer).
