Creates a ring buffer structure operating on the given dataspace.

Creates a ring buffer structure operating on the given dataspace. This assumes a shared ownership of the given dataspace, and keeps a strong reference to it.

Parameters
dataspaceThe dataspace to wrap a ring buffer interface on. (Shared ownership)
modeEither RB_READONLY or RB_WRITEONLY.
Returns
An empty ring buffer on success (Passes ownership), NULL otherwise.
