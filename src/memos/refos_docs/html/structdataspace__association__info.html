Detailed Description

File server CPIO dataspace association.

File server dataspace association structure. It provides look up of a fileserver dataspace using an index. This is used for both window ――▶ dataspace associations, and external dataspace ――▶ dataspace associations. Window ――▶ dataspace associations are used in pager mode, to quickly find the mapped dataspace for a given windowID. External dataspace ――▶ dataspace associations are used in content-init mode, to quickly find the internal dataspace given an external dataspaceID.
