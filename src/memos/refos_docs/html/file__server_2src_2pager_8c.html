Detailed Description

CPIO Fileserver pager RAM frame block module.

Creates big anon RAM dataspace and uses it to manage allocation/deallocation of the frames of the RAM dataspace to be used for paging clients. The frame block works by using a frame pool (cpool_t structure), with the big dataspace mapped into a window. We then call data_datamap in order to page frames into faulting clients, after initialising them with the requested CPIO contents.
