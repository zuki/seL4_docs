RefOS  2.0
Multi-server operating system on seL4
コードドキュメント

摘要

RefOSはseL4上に構築されるシンプルなマルチサーバOSの実装サンプルです。

RefOSの目的は以下の通りです。

  • seL4上に構築されるシンプルなマルチサーバオペレーティングシステムのリファレンス設計を提供する
  • seL4プロトコルの使用方法の実装例を提供する
  • seL4上に構築されるユーザランドプログラムの開発と移植を容易にする
  • 新しいコンポーネントベースのseL4プロジェクトを開始するためのサンプルコードベースとして機能する

RefOSの特徴

  • シンプルで軽量なオペレーティングシステム
  • マルチサーバ・コンポーネントベースの設計
  • プロセスとスレッドの抽象化
  • データスペースの抽象化
  • 共有メモリのサポート
  • 動的なmmapとsbrkのサポート
  • ユーザランドのシリアルドライバとタイマドライバ
  • サンプルユーザプログラム:
    • Terminal - シンプルなターミナルプログラム
    • Tetris - Micro Tetrisの移植版
    • Snake - スネークゲーム

RefOSはあくまでもサンプルシステムであり、進行中であり、実際のオペレーティングシステム ソリューションとして配備されべきものではないことに注意してください。RefOSは検証されて おらず、高い保証もなく、スピードと効率のために最適化されておらず、配備可能なオペレーティング システム環境として期待される機能のほとんどを欠いている可能性があります。
