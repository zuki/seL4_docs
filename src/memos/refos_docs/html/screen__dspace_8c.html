Detailed Description

Timer dataspace interface functions.

This module implements a subset of the dataspace interface (<refos-rpc/data_server.h>), for screen devices. The common dataspace dispatcher module delegates calls to us if it has decided that the recieved message is a timer dataspace call.

This is a thin layer basically wrapping the device_screen module, which has the concrete implementations.
