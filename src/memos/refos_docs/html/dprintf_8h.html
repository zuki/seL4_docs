Detailed Description

Common RefOS debugging output helper functions.

This file defines shared common RefOS debugging output functions. Using a common debugging interface ensures a consistent debug output format, and a centralised control to turn them on or off.

inlinestatic

Debug printf that uses seL4_DebugPutChar, and not dependent on c-library printf.

This debugging function is useful for outputting debugging information beforely using seL4_DebugPutChar() without going through stdio; which is convienient for when stdio is not yet set up. Requires debug kernel to be enabled, does nothing on release builds.

Parameters
fmtThe printf format string.
