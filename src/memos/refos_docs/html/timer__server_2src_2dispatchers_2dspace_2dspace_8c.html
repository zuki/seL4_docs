Detailed Description

Common dataspace interface functions.

This module implements the actual functions defined in <refos-rpc/data_server.h>, and then analyses the parameters to decide which dataspace to delegate the message to. If the parameters indicate the message should be handed off to the serial / timer dataspace, this module then calls the corresponding dataspace interface function of the correct serial / timer dataspace.
