Detailed Description

Helper functions for the data interface.

This file contains a simple layer of helper functions that make using the dataspace interface much easier, but are too complex to have been generated by the stub generator.

Helper function to open a dataspace, allocate a window to map into, then map it there.

Parameters
sessionThe client connection session to the dataspace server. (No ownership)
nameThe name of the dataspace server.
flagsThe read / write / create flags.
modeThe mode to create new file with, in the case that a new one is created.
sizeThe size of mapping window.
dspaceSizeThe size of dataspace to open. Note that some data servers may ignore this.
Returns
A data_mapping_t structure containing the result mapping. Look in the err member of the structure to check for any errors that have occured. (Gives ownership)

Helper function for data_provide_data_from_parambuffer().

Helper function for data_provide_data_from_parambuffer(), that basically copies the given content into the given parameter buffer before calling data_provide_data_from_parambuffer().

Parameters
sessionThe client connection session to the dataspace server. (No ownership)
dspace_fdThe cap to the remote dataspace to provide the content for.
offsetThe offset into the remote dataspace to provide content for.
contentThe content buffer. (No ownership)
contentSizeThe size of the content. (May have maximum content size.)
paramBufferThe parameter buffer that has been set up.
Returns
ESUCCESS if success, refos_error error code otherwise.
