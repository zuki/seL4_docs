Helper function to check for an interface.

Most of the other check_dispatcher_*_interface functions use call this helper function, that does most of the real work. It generates a usable userptr containing the client_t structure of the calling process. If the calling syscall label enum is outside of given range, DISPATCH_PASS is returned.

Parameters
mThe recieved message structure.
userptrOutput userptr containing corresponding client, to be passed into generated interface dispatcher function.
labelMinThe minimum syscall label to accept.
labelMaxThe maximum syscall label to accept.
