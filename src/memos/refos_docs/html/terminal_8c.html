Detailed Description

A simple terminal program for RefOS.

This is a rather minimalistic and hacky terminal app for RefOS. A better console such as bash, or ash, may be ported in the future. The terminal app runs as the first userland program, and presents an interactive shell. It's possible to launcher another instance of the terminal program using the terminal program.
