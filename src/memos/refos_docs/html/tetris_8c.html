Detailed Description

RefOS port of Micro Tetris.

RefOS Tetris

A classic Tetris clone, heavily based on (a port of) Micro Tetris by Joachim Nilsson joach.nosp@m.im.n.nosp@m.ilsso.nosp@m.n@vm.nosp@m.linux.nosp@m..org, which itself is based on the 1989 IOCCC Best Game entry "An obfuscated tetris" by John Tromp john..nosp@m.trom.nosp@m.p@gma.nosp@m.il.c.nosp@m.om.

Website about original IOCCC entry: http://homepages.cwi.nl/~tromp/tetris.html

Website about Micro Tetris: http://freecode.com/projects/micro-tetris http://troglobit.com/tetris.html
