Detailed Description

Process control block structure.

It stores process related information. It is able to own up to PROCESS_MAX_THREADS threads as well as having sessions with up to PROCESS_MAX_SESSIONS servers. authenticateEP is for the process server to notify death of a client when this pcb is for a server.
